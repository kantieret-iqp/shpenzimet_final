2
Extensions → Apps Script → fshij gjithçka → ngjit kodin → Ctrl+S
Pastaj: Deploy → New deployment → Web app → Execute as: Me → Who has access: Anyone → Deploy
Nëse shfaqet paralajmërim → Advanced → Go to app (unsafe) → Allow
Kopjo Web app URL.
3
Ngjit URL-in dhe vendos fjalëkalimet. Gati!
